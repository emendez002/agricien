GNSS Settings Reference

This page provides a complete reference of all settings related to GNSS, GPS, and location in Mapit GIS. Settings are organised by the screen where they appear.

  • Android
  • iOS

External GNSS Settings​

Found in Settings > External GNSS.

These settings control how Mapit GIS connects to external positioning hardware. Only one connection method can be active at a time.

Bluetooth Connection​

SettingTypeDefaultDescription
Paired Bluetooth GPSToggleOffEnable Bluetooth GNSS connection
Selected deviceDevice picker-The paired Bluetooth device to connect to (MAC address and name)

TCP/IP Connection​

SettingTypeDefaultDescription
TCP GPSToggleOffEnable TCP/IP GNSS connection
ServerTextlocalhostTCP server hostname or IP address
PortText52003TCP server port number

Third-Party Companion Apps​

SettingTypeDefaultDescription
Eos Tools ProToggleOffReceive positions from Eos Arrow receivers via Eos Tools Pro
Zeno ConnectToggleOffReceive positions from Leica Zeno receivers via Zeno Connect
Trimble Mobile ManagerToggleOffReceive positions from Trimble receivers via Mobile Manager
Trimble SPaceToggleOffReceive positions from Trimble DA2 via SPace
SXblue ToolboxToggleOffReceive positions from Geneq SXblue receivers via SXblue Toolbox

Tracking Settings​

Found in Settings > Miscellaneous under the Tracking category.

These settings control GPS track recording behaviour.

SettingTypeDefaultRangeDescription
Show track recording buttonToggleOn-Show/hide the track recording button on the map toolbar
Minimum time between updatesSlider2 seconds0-60 sMinimum time interval between recorded track points
Minimum distance between updatesSlider2 metres0-100 mMinimum distance interval between recorded track points
tip

For vehicle surveys, increase the time interval (5-10 s) and distance (10-50 m) to avoid excessive point density. For walking surveys on foot, the defaults (2 s, 2 m) work well.

Settings Not in the UI​

Some location-related values are managed internally and not directly exposed in the settings screens:

ValueDefaultDescription
Center on GPSOffWhether the map follows the GPS position - toggled via the map toolbar GPS button
Location providerInternal GPSResolved automatically based on which connection toggles are enabled
iOS Limitation

On iOS, external GNSS connections are limited to TCP/IP only. Bluetooth, companion app, and rangefinder settings are not available. Geoid model selection is also not yet supported.
