As a person who uses $12000 data collection devices daily for work I can say this app works very well. It's simple to use if you follow the directions. So far has worked great for personal use when I want to collect a few items such as wild flowers and trees of interest on my phone while doing more complicated collections. Miroslav Kocic​

So, let me start off by saying I am using the full version. This is a brilliant app! Great job. As an archaeologist who does a lot of field walking, and uses a wide variety of data recorders and software to go with it, I find this app much better than a lot of more expensive competitive product.

Mike Miller​

This is an absolutely fabulous app for GIS data collection on a budget. You can't get as fancy with the input forms as you could with a XLS forms based app, but it fits the bill for most simple applications and doesn't require an expensive user account. Just collect the data and export a Shapefile or Geojson. Exactly what I was looking for my application in a poor area in West Africa where nobody is going to be able to get an ArcGIS online subscription. I've been working with GPS for over 2 decades. I've used almost every GPS mapping product out there, and it's clear that the development of this one has come from the perspective of an actual user and not a marketing department. Very happy.

Dan Cuillin​

Excellent This app does pretty much exactly what is required of a GIS data collection tool. Satellite basemaps, dropdown or value list form filling, custom forms, export options, GPS or cursor input and now layer import and WMS overlays. I’ve been through the other similar apps, and they all miss one or more of the above (even the expensive ones). This one is the only app to have them all, and in a simple easy to use format. Nice work.

Marko Beslic​

Probably the best application for GIS/GPS data collection. I tried about 20 applications and this is the best for data collection in the field. App is intuitive and easy to use. Adding points is possible by GPS or manually by positioning on the map, which is great if the exact position is unreachable. Feature attributes can be quickly selected using the drop-down menu, which speeds up data collection. Easy data exchange with GIS software using the import/export functions in various formats like CSV, KML...
